# STA302 Detailed Notes

## Overview
This repository contains detailed notes for **STA302 – Methods of Data Analysis**. The material is organized to support understanding of core statistical concepts, with a focus on both theory and application.

## Contents
- Simple Linear Regression (SLR)
- Multiple Linear Regression (MLR)
- Hypothesis Testing
- Confidence Intervals
- ANOVA (Analysis of Variance)
- Lack of Fit Testing
- Model Diagnostics
- Residual Analysis
- Model Selection (AIC, Adjusted R², etc.)

## Structure
The notes are written in a clear, structured format and are intended for:
- Exam preparation
- Concept reinforcement
- Quick reference during problem solving
